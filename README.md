# ubuntu

Ubuntu Useful Commands

`Update Ubuntu`

`CMD: sudo apt-get update`

`Upgrade Ubuntu`

`CMD: sudo apt-get upgrade`

`Change Directory `

`CMD: cd [Directory]`

`Install Apache2`

`CMD: sudo apt-get install apache2`

`Start Apache2`

`CMD : service apache2 start`

`Status Apache2 `

`CMD: service apache2 status`

`Stop apache2`

`CMD: service apache2 stop`
